From installation to fine tuning

YaST is both an extremelly flexible installer and a powerful control center. One tool to setup them all.

Tweak everything

YaST includes modules to configure almost every single aspect of a Linux system... and provides a solid framework to develop even more.

Three full-featured interfaces

YaST offers two graphical interfaces (Gtk+ and Qt) and a terminal-based one, perfect for remote administration.

For newbies and experts

YaST empowers system administrators not by hidding the underlying system details but by providing nice ways to handle them.

Install automatically

AutoYaST enables you to export system configuration settings and then uses them to install thousands of other systems automatically.
